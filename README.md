
node.js
### 코드 받기
```
git clone https://github.com/HeegyuKim/simple-nodejs-blog.git
```

### 1. 패키지 설치
```
npm install
```
### 2. 데이터베이스 생성 및 샘플 데이터 넣기 
```
sqlite3 blog.db < schema.sql
sqlite3 blog.db < sample_dumps.sql
```
### 3. 실행 및 접속
```
node blog.js
```
구름 IDE가 아닌 자기 컴퓨터에서 하시는 분들은
http://localhost:3000/ 으로 접속하면 됩니다.
